Swift3.0速查手册 values
字符串通过 + 拼接 
print("swift" + "lang")                  // swiftlang

print(1 + 1)                             // 2
将Int类型转化成字符串 
print("1+1 = " + String(1 + 1))          // 1+1 = 2

print(7.0 / 3.0)                         // 2.33333333333333

print(String(7.0) + " / " + String(3.0)) // 7.0 / 3.0
布尔值运算 (YES) 
print(true && false)                     // false
print(true || false)                     // true
print(!true)                             // false
